# jetpack

Flutter jetpack poject for you . 

## Running environment

- flutter
- node

## Getting Started

First step, when the environment configuration is complate, open terminal, run the following script.
```
/// step 1
flutter pub get
/// step 2
cd node/server/
npm install
/// step 3
. bin/test_start_node.sh
```
Then, open Chrome and input http://localhost:3000.
